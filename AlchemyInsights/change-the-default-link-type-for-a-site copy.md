# <a name="change-the-default-link-type-for-a-site"></a>變更網站的預設連結類型

使用者可以透過傳送連結，共用 SharePoint 和 OneDrive 中的檔案和資料夾。 他們應根據要授與權限的對象選取連結類型。

如需詳細資訊，請參閱 [變更網站的預設連結類型](/sharepoint/change-default-sharing-link)。
