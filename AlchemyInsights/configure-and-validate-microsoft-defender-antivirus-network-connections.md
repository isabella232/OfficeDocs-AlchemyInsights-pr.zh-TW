# <a name="configure-and-validate-microsoft-defender-antivirus-network-connections"></a>設定及驗證 Microsoft Defender 防毒軟體網路連線

若要確保可以更新 Microsoft Defender 防毒軟體，您必須將網路設定為允許端點與特定 Microsoft 伺服器之間的連線。 如需詳細資訊，請參閱[設定及驗證 Microsoft Defender 防毒軟體網路連線](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/configure-network-connections-microsoft-defender-antivirus) (部分機器翻譯)。
