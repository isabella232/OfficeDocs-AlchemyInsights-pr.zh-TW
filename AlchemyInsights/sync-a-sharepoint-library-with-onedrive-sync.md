# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>使用 OneDrive 同步處理來同步處理 SharePoint 文件庫

若要讓 SharePoint 檔案同步處理到本機硬碟，請完成下列步驟：

1. 瀏覽至您要同步處理到本機硬碟的文件庫。
2. 從功能表選項中選取 [同步處理] 按鈕。
3. 按照逐步步驟完成同步處理。
