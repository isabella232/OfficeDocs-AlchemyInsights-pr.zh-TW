# <a name="service-principal-key"></a>服務主體金鑰

如需服務主體的金鑰認證管理的指導方針，請參閱下列文章：

- [AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) - 使用 PowerShell 為服務主體取得金鑰認證。

- [AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) - 使用 PowerShell 為服務主體建立新的金鑰認證。

- [AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) - 使用 PowerShell 從服務主體移除金鑰認證。
