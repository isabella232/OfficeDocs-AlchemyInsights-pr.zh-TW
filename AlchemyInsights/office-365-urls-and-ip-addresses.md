# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Microsoft 365 URLs 和 IP 位址範圍

Microsoft 365 需要透過指定的 IP 位址（稱為*端點*）連接至網際網路。
如需需要可存取的端點清單，請參閱[Microsoft 365 URLs 和 IP address 範圍](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Microsoft 365 IP 位址和 URL web 服務可協助您更好地識別及區別 Microsoft 365 網路流量，讓您更容易評估、設定和保持最新的變更。 請參閱[Office 365 IP 位址和 URL web 服務](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service)。
