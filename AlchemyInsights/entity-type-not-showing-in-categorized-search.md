# <a name="entity-type-not-showing-in-search-results"></a>搜尋結果中未顯示的實體類型

若要在 Dynamics 365 統一介面中，顯示在分類搜尋或相關性搜尋中的實體類型，必須先將其新增至應用程式模組。

附注：搜尋相關性搜尋中的附件時，必須將附件實體新增至 app 模組，即使父筆記錄是結果中顯示的內容。
