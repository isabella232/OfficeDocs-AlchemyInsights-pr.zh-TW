# <a name="change-your-default-search-engine-in-microsoft-edge"></a>在 Microsoft Edge 中變更您的預設搜尋引擎

若要變更預設搜尋引擎，請遵循下列指示：
1. 在網址列中，使用您想要設為預設值的搜尋引擎執行搜尋。
2. 選取 **... ([設定] 等等) > [設定] > [隱私權]、搜尋和服務**。
3. 一路向下捲動至 [**服務**] 區段，然後選取 [**網址列和搜尋**]。
4. 從 [**在網址列中使用的搜尋引擎**] 功能表，選取您偏好的搜尋引擎。
