# <a name="cutover-migration"></a>完全移轉

使用[完全移轉](https://admin.microsoft.com/adminportal/home#/cutoverwizard)來移轉使用者信箱的內容。 此方法支援最多移動 2000 個信箱，移動擁有成員資格的郵件使用者、郵件連絡人和擁有郵件功能的群組。 由於執行程式所花的時間長度，最佳做法會規定在 150 位以下的使用者群組中移轉。
