# <a name="add-or-remove-a-custom-role"></a>新增或移除自訂角色

若要管理 Azure 資源，請參閱[使用 Azure 入口網站新增或移除 Azure 角色指派](https://docs.microsoft.com/azure/role-based-access-control/role-assignments-portal)。

若要管理 Azure Active Directory (Azure AD) 的自訂角色，請參閱 [Azure Active Directory 中的自訂系統管理員角色 (預覽)](https://docs.microsoft.com/azure/active-directory/users-groups-roles/roles-custom-overview)。
