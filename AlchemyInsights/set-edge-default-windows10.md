# <a name="set-microsoft-edge-as-the-default-browser-on-a-windows-10-device"></a>在 Windows 10 裝置上將 Microsoft Edge 設定為預設瀏覽器

方法如下：

1. 選取 [開始 **]** 按鈕，然後輸入 **預設應用程式**。

1. 在搜尋結果中，選取 [預設應用程式 **]**。

1. 在 [網頁瀏覽器 **]** 下，選取 **Microsoft Edge**。
