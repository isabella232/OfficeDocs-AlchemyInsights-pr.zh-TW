# <a name="adding-an-overlay-to-a-sharepoint-calendar"></a>將疊加新增至 SharePoint 行事曆

SharePoint「行事曆」是傳統清單類型，有時會與 Microsoft 365 小組行事曆混淆。
 
如果您在網站上使用 SharePoint 日曆，您可以覆迭您自己的 Exchange 行事曆，以建立組合的行事曆視圖。 此外，您也可以在相同的網站集合中，將其他 SharePoint 的行事曆疊加，只要這兩 SharePoint 的行事曆都一樣。
 
若要將疊加新增至傳統的 SharePoint 行事曆，請遵循以[Exchange 或 SharePoint](https://support.office.com/article/Overlay-a-SharePoint-calendar-with-a-calendar-from-Exchange-or-SharePoint-4CAEBE59-3994-4A94-9322-B31ABB8A5E9A)中的行事曆覆迭 SharePoint 行事曆中的步驟。
