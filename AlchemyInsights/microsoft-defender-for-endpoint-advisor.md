# <a name="deploy-microsoft-defender-for-endpoint"></a>部署適用於端點的 Microsoft Defender

[Microsoft Defender For Endpoint advisor](https://go.microsoft.com/fwlink/?linkid=2146241)提供說明，可協助您的商業網路防止、偵測、調查和回應高級威脅。 作出明智的組織弱點評估，並決定最佳的部署套件和設定方法。

**附注**：適用于 Defender for Endpoint 的 Microsoft 大量授權是必要的。
