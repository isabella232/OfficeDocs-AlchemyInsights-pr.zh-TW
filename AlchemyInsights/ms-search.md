# <a name="get-started-with-microsoft-search"></a>開始使用 Microsoft 搜尋

使用 Microsoft 搜尋以尋找人員、檔案、組織圖表、網站、常見問題的解答，以及其他可協助您在所使用應用程式中完成工作的內容。

若要深入瞭解，請參閱[Microsoft 搜尋的總覽](https://go.microsoft.com/fwlink/?linkid=2157644)。
