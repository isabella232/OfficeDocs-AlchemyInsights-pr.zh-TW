# <a name="update-ios-device-to-latest-os-version"></a>將 iOS 裝置更新為最新 OS 版本

只有在受監督的裝置和執行 iOS 10.3 及更新版本的 iOS 裝置上，才能使用作業系統版本更新。

如果您的裝置符合版本需求，請確認：  
- 裝置使用者沒有延遲更新。  
- 目前的 iOS 更新原則設定 (例如受限制的時段) 不會阻礙安裝更新。

更新您的 iOS：

1. 登入 Azure 入口網站。
2. 選擇 **[所有服務]** > **[Intune]**。
3. 在 Intune 窗格中，選擇 **[軟體更新]** > **[iOS 版的更新原則]**。
