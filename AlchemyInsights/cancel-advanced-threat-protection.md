# <a name="cancel-advanced-threat-protection"></a>取消高級威脅防護

系統管理員可以在 Microsoft 365 系統管理中心取消其組織的高級威脅防護訂閱。

1. 移至 [**帳單**  >  [產品](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面。
2. 在 [ **產品** ] 索引標籤上，尋找您要取消的高級威脅防護計畫。 選取 [ (更多動作) ] 的三個點，然後選取 [ **取消訂閱**]。
3. 在 **取消訂閱** 窗格中，選擇取消的原因。 您也可以選擇提供任何意見反應。
4. 選取 **[儲存]**。

**退款：** 任何按比例計算的款項將會在下個計費週期內退回。

若要深入瞭解，請參閱 [取消您的訂閱](/microsoft-365/commerce/subscriptions/cancel-your-subscription)。
