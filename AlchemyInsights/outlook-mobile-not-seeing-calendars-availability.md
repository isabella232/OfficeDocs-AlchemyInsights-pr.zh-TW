# <a name="im-not-seeing-the-calendars-i-expect-when-choosing-my-availability"></a>我在選擇可用時看不到預期的行事曆

如果您看不到預期的行事曆，請移至 [行事曆]，並在螢幕的左上角點擊 [功能表] 按鈕，並確定已選取您期望看到的行事曆。 選擇 [傳送可用性] 的時間時，只會顯示此功能表中啟用的行事曆。
