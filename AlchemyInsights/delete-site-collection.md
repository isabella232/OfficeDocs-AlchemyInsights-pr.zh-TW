# <a name="delete-a-sharepoint-site"></a>刪除 SharePoint 網站

1. 在 [新增 SharePoint 系統管理中心] 的 [網站] 頁面上，按一下您要刪除之網站旁的核取方塊。
    
2. 在功能區工具列上，按一下 [ **刪除**]。
