# <a name="view-or-modify-a-journal-rule"></a>檢視或修改日誌規則

方法如下：

1. 在 [Exchange 系統管理中心](https://go.microsoft.com/fwlink/p/?linkid=2059104)，移至 [**規範管理**] [  >  **日誌規則**]。
2. 在清單檢視中，您將看見組織中所有的日誌規則。
3. 連按兩下要檢視或修改的規則。
4. 在 [ **日誌規則**] 中，修改所需的設定。
