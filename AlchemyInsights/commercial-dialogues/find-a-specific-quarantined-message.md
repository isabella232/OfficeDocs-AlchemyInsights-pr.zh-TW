# <a name="find-a-specific-quarantined-message"></a>尋找特定隔離郵件

1. 移至 [安全性 & 規範中心](https://go.microsoft.com/fwlink/p/?linkid=2077143)。
2. 移至 **威脅管理**  >  **回顧**  >  **隔離**。
3. 選擇 [ **搜尋**]。
4. 選取條件旁的核取方塊來選擇任何組合， (您此時無法使用萬用字元) 。
