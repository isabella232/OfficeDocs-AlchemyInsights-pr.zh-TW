# <a name="problem-verifying-a-domain"></a>驗證網域時發生問題

如果 Office 365 未驗證您的網域一小時或兩次，有問題。

在下列提示中選擇 [ **否** ] 表示 **我未解決您的問題** ，然後您會看到與人類交談的選項。
