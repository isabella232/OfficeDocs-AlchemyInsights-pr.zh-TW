# <a name="i-dont-see-automatic-replies"></a>我沒有看到自動回復

[！注意] 如果您 **在選取 [** 檔案] 後沒有看到 [自動回復]，表示您使用的 Gmail、Yahoo 或其他 POP 或 IMAP 帳戶不支援 Outlook 自動回復功能。 您可以設定會回復內送郵件的規則，但只有在您留下 Outlook。
