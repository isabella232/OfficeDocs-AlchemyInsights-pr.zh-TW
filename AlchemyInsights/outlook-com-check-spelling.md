# <a name="use-your-browser-to-check-spelling"></a>使用瀏覽器檢查拼寫

雖然 Outlook .com 沒有拼寫檢查功能，但許多網頁瀏覽器都會自動檢查拼寫，包括 Microsoft Edge、Internet Explorer 10 和更新版本，以及目前的 Firefox、Chrome 和 Safari 版本。 請檢查網頁瀏覽器的選項，以深入瞭解如何檢查拼寫。
