# <a name="skype-icon-missing"></a>Skype 圖示遺失

我們暫時從 Outlook 中移除 Skype 圖示做為設計重新整理和功能更新的一部分，我們非常高興。我們建議您耐心等候，我們為您準備新的和改善的體驗。

您仍然可以在[web.skype.com](https://web.skype.com/)上透過瀏覽器使用 Skype。
