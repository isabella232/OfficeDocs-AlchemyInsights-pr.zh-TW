# <a name="fix-problem-viewing-protected-message-in-outlook"></a>修正此問題在 Outlook 中查看受保護的郵件

在某些情況下，如果已使用資訊版權管理來保護郵件，Outlook 可能無法開啟從其他 Microsoft 365 組織中的使用者傳送的郵件。

若要開啟和查看這類受保護的郵件，請執行下列步驟：

1. 確定您已安裝 Outlook 1808 + 或更新版本。 若要尋找您的版本號碼，請參閱 [我使用哪一個版本的 Office？](https://support.microsoft.com/office/about-office-what-version-of-office-am-i-using-932788b8-a3ce-44bf-bb09-e334518b8b19)
2. 嘗試使用網頁上的 Outlook 來查看電子郵件： [outlook 網頁版登錄](https://outlook.office365.com/mail/inbox)

> [!NOTE]
> 我們正在努力修正 Outlook 2016 和2019中的問題。 它不會在 Outlook 2010 或2013中修復。
