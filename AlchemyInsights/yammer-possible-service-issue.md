# <a name="yammer---possible-service-issue"></a>Yammer-可能的服務問題

可能是暫時性的問題，使用者在存取 Yammer 時可能會發生錯誤。 請檢查 [服務健康情況儀表板](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) ，查看是否會影響您的網路。
