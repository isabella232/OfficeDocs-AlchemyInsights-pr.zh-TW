# <a name="cant-delete-files-or-photos-from-files-in-outlook-on-the-web"></a>無法從 Outlook 網頁版中的檔案刪除檔案或相片

**Files** 會顯示附加至信箱中郵件的檔案和相片。 若要從檔案中移除檔案和相片，您必須刪除其所附加的電子郵件。 在刪除郵件之後，您也需要從 [刪除的郵件] 資料夾中刪除郵件。
