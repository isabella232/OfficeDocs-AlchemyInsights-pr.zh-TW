# <a name="make-bingcom-the-default-home-page"></a>讓 Bing.com 成為預設首頁

若要在 Microsoft Edge、Google Chrome 和 Internet Explorer 中 Bing 預設首頁，請依照[將 Bing .com 設定為預設首頁](https://go.microsoft.com/fwlink/?linkid=2149816)的指示進行。
