# <a name="add-contacts-to-speed-dial"></a>將連絡人新增至快速撥號

1. 在 [  **通話**] 中，移至 [  **連絡人**]。
2. 如果您看到要新增至快速撥號之人員的卡片，請選取 [  **更多選項**  ] (三個點) 在其卡片上，然後選取 [  **新增至快速撥號**]。

如果您看不到人員，請選取螢幕右上角的 [  **新增快速撥號**  ]，然後輸入其名稱，選取人員，然後選取 [  **新增**]。
