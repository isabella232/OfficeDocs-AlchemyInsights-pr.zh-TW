# <a name="banned-sending-ip"></a>禁止傳送 IP

傳送給 Office 365 收件者時，寄件者可能會遇到此錯誤。 寄件者必須造訪[Office 365 Anti-Spam IP 取消列出入口網站](https://sender.office.com/)，並提供其電子郵件地址和必須 delisted 的 ip 位址。 如果他們沒有從入口網站接收驗證電子郵件，請讓他們檢查 [垃圾郵件] 資料夾和/或嘗試使用不同的電子郵件地址。 

- 如需詳細指示，請參閱 [使用取消列出入口網站將您自己從封鎖的寄件者清單中移除](https://docs.microsoft.com/microsoft-365/security/office-365-security/use-the-delist-portal-to-remove-yourself-from-the-office-365-blocked-senders-lis?view=o365-worldwide)。
