# <a name="upload-files-to-onedrive-or-sharepoint"></a>Upload 檔案 OneDrive 或 SharePoint

您可以在 SharePoint Online 中將檔案上傳至文件庫，只需要將檔案從您的電腦拖曳，然後放入文件庫中。 透過 OneDrive，您可以在電腦與雲端之間同步處理檔案，讓您可以從任何位置取得檔案。 您也可以直接在檔案資源管理器中使用已同步處理的檔案，甚至在離線時存取檔案。 如需上傳及同步處理的詳細資訊，請造訪下列連結。

- [Upload 資料夾或檔案至文件庫](https://support.office.com/article/upload-a-folder-or-files-to-a-document-library-eb18fcba-c953-4d45-8d90-8da66edeacdb)

- [使用 sync Upload](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-in-windows-615391c4-2bd3-4aae-a42a-858262e42a49)

- [檔限制 (無效字元、檔案上傳大小、路徑長度等等。) ](https://support.office.com/article/invalid-file-names-and-file-types-in-onedrive-onedrive-for-business-and-sharepoint-64883a5d-228e-48f5-b3d2-eb39e07630fa)

- [修正 OneDrive 同步處理問題](https://support.office.com/article/Fix-OneDrive-sync-problems-83ab0d8a-8400-45b0-8dcf-dc8aa8a6bcf8)

- [修正在 OneDrive 網站上傳檔案的相關問題](https://support.office.com/article/Fix-problems-uploading-files-on-the-OneDrive-website-9afcc4a0-e344-4bc9-9c9d-59d3e802247e)

- [盡可能提升上傳和下載速度](https://support.office.com/article/Maximize-upload-and-download-speed-8eeadfb8-501f-406d-997b-98ab6ff67f43)
