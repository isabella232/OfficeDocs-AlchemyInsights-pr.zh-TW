# <a name="configure-multifactor-authentication"></a>設定多重要素驗證

若要設定 MFA，請參閱[設定多重要素驗證](/microsoft-365/admin/security-and-compliance/set-up-multi-factor-authentication)或使用[Microsoft 365 系統管理中心設定指南](https://admin.microsoft.com/AdminPortal/Home?ref=/modernonboarding/mfasetupguide)。

**附注**：您必須是全域系統管理員，才能啟用和設定 MFA。
