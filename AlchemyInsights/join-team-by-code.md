# <a name="join-a-team-by-code"></a>依代碼加入小組

Microsoft 已發現使用者無法建立 **[加入小組]** 代碼，或使用 **[使用代碼加入小組]** 的功能來加入小組。如需因應措施，小組擁有者可以視需要手動新增使用者。
