# <a name="activate-and-assign-software-as-a-service-app-licenses"></a>啟動並指派軟體即服務應用程式授權 

## <a name="to-activate-apps"></a>啟動應用程式

1. 在系統管理中心中，移至 [**帳單**  >  **[產品](https://go.microsoft.com/fwlink/p/?linkid=842054)**] 頁面，然後選擇 [**應用程式**] 索引標籤。

2. 尋找並選取您要管理的應用程式。

3. 在 [ **授權**] 底下，選取 [ **設定帳戶**]。  

## <a name="to-assign-app-licenses"></a>指派應用程式授權

1. 在系統管理中心中，移至 [**帳單**  >  **[產品](https://go.microsoft.com/fwlink/p/?linkid=842054)**] 頁面，然後選擇 [**應用程式**] 索引標籤。

2. 尋找並選取您要管理的應用程式。  

3. 在 [**設定 & 動作**] 下，選取要 **在 publisher 入口網站中管理** 的連結。
