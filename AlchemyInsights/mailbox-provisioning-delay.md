# <a name="your-users-mailbox-is-being-created"></a>正在建立使用者的信箱

最多可能需要一小時，新的信箱才可供使用。
  
如果使用者的信箱已 "準備" 超過一小時，您可以嘗試刪除然後重新建立使用者，或使用「讓我們撥打您」選項與支援工程師交談。
