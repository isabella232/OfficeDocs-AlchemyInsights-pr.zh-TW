# <a name="change-billing-country-or-region"></a>變更帳單的國家或地區

不可能從 Microsoft 365 系統管理中心中的 **計費**  >  **計費帳戶** 變更國家/地區。 您必須註冊新帳戶，然後選擇所需的國家/地區。
