# <a name="conversion-of-microsoft-365-group-and-shared-mailboxes"></a>Microsoft 365 群組和共用信箱的轉換

目前沒有任何方法可以在 Microsoft 365 群組信箱與共享信箱之間進行轉換。

請 [在這裡](https://aka.ms/M365GroupToShared)協助您投票這項功能。
