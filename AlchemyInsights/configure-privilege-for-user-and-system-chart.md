# <a name="configure-privilege-for-user-and-system-chart"></a>設定使用者和系統圖表的權限

**症狀**：沒有圖表的讀取權限。

**原因**：權限不足或未持有必要權限的錯誤。

**解決方案**：設定圖表的權限

**步驟**：

1. 移至 [設定] -> [自訂]-> [自訂系統] > [安全性角色 **]**。

2. 連按兩下該角色。

    - [使用者圖表] 權限設定位於 [核心記錄 **]** 索引標籤底下。
    - [系統圖表] 權限設定位於 [自訂 **]** 索引標籤下。
