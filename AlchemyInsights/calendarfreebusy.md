# <a name="issues-seeing-calendar-freebusy-information"></a>查看行事曆空閒/忙碌資訊的問題

如果您在其他的 Outlook 日曆中看不到行事曆空閒/忙碌資訊，或是其他人無法在您的行事曆中看到空閒/忙碌資訊，請使用[Microsoft 支援及修復小幫手進行 Microsoft 365](https://diagnostics.office.com/)。
