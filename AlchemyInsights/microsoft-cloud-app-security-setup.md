# <a name="microsoft-cloud-app-security-setup"></a>Microsoft Cloud App Security 設定

設定 [Microsoft 雲端 App 安全性](https://aka.ms/cloudappsecuritysetup)，以改善雲端活動的可見度，並增加貴組織的資料保護。 控制資料移動、雲端應用程式和分析，協助防止整個雲端服務的威脅。
