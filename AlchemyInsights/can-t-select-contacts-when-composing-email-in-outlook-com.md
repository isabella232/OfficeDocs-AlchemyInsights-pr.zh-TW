# <a name="cant-select-contacts-when-composing-email"></a>無法在撰寫電子郵件時選取連絡人

在撰寫新的電子郵件時，有些客戶按一下 [ **收件者** **] 或 [** 副本] 按鈕時，看不到他們的連絡人清單。 若要選取連絡人，請從 [ **收件者** **] 或 [** 副本] 欄位開始輸入，並顯示相關的連絡人。
