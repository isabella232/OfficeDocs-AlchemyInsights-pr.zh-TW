# <a name="control-creation-of-microsoft-365-groups"></a>控制項的 Microsoft 365 群組的建立

根據預設，公司內的所有使用者都可以建立群組。 如有需要，您可以變更此設定，並將建立群組的許可權指派給公司內的一組人員。 請注意，這會對從任何 Microsoft 365 服務 (（如 Outlook、Teams、Planner 等）建立群組產生影響) 。
  
指示：[管理誰可以建立 Microsoft 365 群組](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
