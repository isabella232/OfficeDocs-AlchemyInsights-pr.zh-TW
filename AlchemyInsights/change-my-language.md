# <a name="change-my-language"></a>變更我的語言

Outlook 應用程式會自動轉譯成您的裝置設定的語言。 

 (會看到[Outlook 中支援的語言清單](https://acompli.helpshift.com/a/outlook/?s=general-questions&f=in-which-languages-is-your-app-translated)。 )  

**若要變更裝置上的語言**： 

**在 Android 上**： 

- 裝置設定 > 語言和輸入 

**IOS**： 

- 裝置設定 > 一般 > 語言 & 地區
