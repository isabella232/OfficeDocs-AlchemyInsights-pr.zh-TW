# <a name="already-paid"></a>已經支付？

如果您已送出付款，您可以讓我們知道：
  
1. 在系統管理中心中，前往 [**帳單**] \> [[訂閱](https://go.microsoft.com/fwlink/p/?linkid=842054)] 頁面。

2. 選擇訂閱，然後在 [ **我的帳單** ] 區段中，選擇 [ **查看帳單**]。

3. 在 [ **帳單詳細資料** ] 頁面上，選擇 [ **已支付]？** 系統會要求您輸入與所傳送之付款相關的資訊。

4. 輸入所有必要資訊之後，請選擇 [ **傳送**]。
