# <a name="merge-microsoft-accounts"></a>合併 Microsoft 帳戶

- 您無法合併或連結 Microsoft 帳戶和 Office 365 公司或學校帳戶，因為這兩者位於不同的服務中，但這兩個帳戶可以並用。
- 您無法合併或連結 Microsoft 帳戶 (即使是個別的電子郵件地址)，或非 Microsoft 帳戶 (例如 Gmail 帳戶) 到現有的 Microsoft 帳戶。
