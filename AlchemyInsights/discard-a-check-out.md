# <a name="discard-a-check-out-from-a-document-library"></a>捨棄從文件庫檢出

1. 以滑鼠右鍵按一下您要捨棄取出的檔案。
    
2. 按一下 [ **更多**]，然後按一下 [ **放棄取出**]。 
    
3. 在對話方塊中，按一下 [ **放棄取出** ] 以清除 [取出]。 
    
如需有關存回及取出的其他問題，請參閱 [關於簽出、存回及版本的主要問題](https://go.microsoft.com/fwlink/?linkid=2018786)。
