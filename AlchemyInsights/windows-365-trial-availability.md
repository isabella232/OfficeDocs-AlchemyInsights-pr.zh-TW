# <a name="windows-365-trial-availability"></a>Windows 365 試用版可用性

由於需求很大，我們已暫時暫停新的 Windows 365 試用版訂閱。

我們鼓勵您註冊，以在試用版恢復時收到通知。 如需註冊，請參閱[註冊以深入了解 Windows 365](https://aka.ms/Win365InfoNotification)。
