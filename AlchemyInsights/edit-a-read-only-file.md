# <a name="edit-a-read-only-file"></a>編輯唯讀檔案

在文件庫中，當您按一下檔的名稱時，Office 程式可能會意外地將其開啟為唯讀檔案。 若要將其開啟以進行編輯，請按一下 [ **開啟** (三個點) ]，然後按一下 [ **編輯]。**
