# <a name="export-user-account-information"></a>匯出使用者帳戶資訊

若要匯出 Microsoft 365 系統管理中心中的使用者帳戶資訊，包括授權、密碼原則、封鎖帳戶的資訊、AAD 同步狀態等等，請移至 [**使用者** 作用中  >  [使用者](https://go.microsoft.com/fwlink/p/?linkid=834822)] 頁面，然後選取 [**匯出使用者**]。
