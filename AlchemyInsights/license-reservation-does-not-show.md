# <a name="license-reservation-does-not-show"></a>沒有顯示授權保留

- 授權會在 24 到 48 小時內佈建或啟用。

- 如果保留授權來自大量授權企業合約，請參閱[如何與 Microsoft 大量授權服務中心連絡](https://support.microsoft.com/help/4471406/how-to-contact-the-microsoft-volume-licensing-service-center)。
