# <a name="invalid-file-and-folder-names-in-onedrive-and-sharepoint"></a>OneDrive 和 SharePoint 中的檔案和資料夾名稱無效

在檔案和資料夾名稱中不允許這些字元 " \* ： \< \> ？ / \ | 
  
有些組織還沒有啟用 # 和% 的支援。 若要瞭解如何在組織中允許這些字元，請參閱 [啟用 # 和% Support](https://go.microsoft.com/fwlink/?linkid=862611)。
